# 🏢 Condomínio API (Backend)

Sistema de gestão condominial desenvolvido para controle financeiro, automação de rateio de despesas e administração de unidades.

## 🚀 Tecnologias
- **Node.js** (Runtime)
- **Express** (Framework Web)
- **Prisma** (ORM)
- **PostgreSQL** (Banco de dados)

## 📁 Estrutura de Pastas
- `src/controllers`: Lógica de controle para Condomínios, Financeiro, Usuários e Rateio.
- `src/services`: Regras de negócio complexas como automação de cobranças e cálculos de balancete.
- `src/routes`: Definição de endpoints da API.
- `prisma/`: Schema do banco de dados e migrações.
